# Mobile Sign-Up Flow UI/UX Design

This repository contains the UI/UX design for a mobile sign-up flow, created using Figma.

## Description

The design focuses on:
- **User-Friendly Interface**
- **Accessibility**
- **Aesthetics**
- **Efficiency**

## Screens

- **Welcome Screen**
- **Sign-Up Form**
- **Verification Screen**
- **Confirmation Screen**
